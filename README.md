# 养老院管理系统 java161

## 项目概述

养老院管理系统是基于SpringBoot和Vue的前后端分离系统，旨在提供全面的管理功能，确保养老院日常运营的高效与便捷。本系统包含源码、数据库及详尽的文档。

## 技术栈

- **开发环境**：Windows 10
- **架构模式**：MVC/前后端分离
- **主要技术**：
  - JDK版本：Java JDK 1.8
  - 开发工具：IDEA
  - 数据库版本：MySQL 8.0
  - 数据库可视化工具：Navicat
  - 服务器：SpringBoot自带的Apache Tomcat
  - 前后端技术：Java, SpringBoot, MyBatis, MySQL, jQuery, HTML, Vue
